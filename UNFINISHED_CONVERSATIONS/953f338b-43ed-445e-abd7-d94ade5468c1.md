**Vitest** is a fast unit testing framework for JavaScript/TypeScript, built on top of Vite.

```typescript
// Example vitest test
import { describe, it, expect } from 'vitest'

describe('math', () => {
```

- **Jest-compatible API** - familiar syntax (describe/it/expect)
- **Native TypeScript** - no config needed for TS
- **Works with Bun** - PAI uses Bun, vitest runs on it

**vs Jest:**
| | Vitest | Jest |
|---|--------|------|
| Speed | Faster (Vite-powered) | Slower |
| TS support | Native | Needs ts-jest |
| Config | Minimal | More setup |
| API | Same as Jest | Standard |
