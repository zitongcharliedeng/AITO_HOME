## ðŸ“š What is Vitest?

**Vitest** is a blazing-fast unit testing framework for JavaScript/TypeScript, built natively on **Vite**.

**Key features:**
- **Vite-native** - uses Vite's transform pipeline, so your tests share the same config as your app
- **Jest-compatible API** - drop-in replacement for most Jest tests (`describe`, `it`, `expect`, etc.)
- **ESM-first** - native ES modules support out of the box
- **TypeScript support** - works with TS without extra config
- **Hot Module Replacement (HMR)** - tests re-run instantly on file changes
- **Built-in coverage** - via c8/istanbul
- **Snapshot testing** - like Jest
- **In-source testing** - can write tests alongside your code

**Why use Vitest over Jest?**
- Significantly faster (especially in watch mode)
- No duplicate config if you already use Vite
- Better ESM support
- Modern architecture

**Basic example:**
```typescript
import { describe, it, expect } from 'vitest'

describe('math', () => {
  it('adds numbers', () => {
    expect(1 + 1).toBe(2)
  })
})
```
